# Decision: Build Rhythm — 15 Hours/Week

**Date:** April 16, 2026  
**Decided by:** Anti

## Decision

- **5 hours** across weekdays (1h/day)
- **10 hours** on weekends
- **Total:** 15 hours/week
- **Timeline:** ~13 weeks (~3 months) from zero to Phase 4 complete

## Phase timeline estimates

| Phase | Weeks |
|---|---|
| Phase 1 — Foundation | ~3 |
| Phase 2 — Context & Intelligence | ~4 |
| Phase 3 — Autonomous Agent | ~3 |
| Phase 4 — Learning System | ~3 |
| **Total** | **~13 weeks** |

## Ground rules

1. Finish each phase before the next — no "80% done" skipping
2. No parallel phases
3. Use Wingman for 2 weeks between phases before adding the next layer
4. Commit after every working feature
5. Out-of-scope ideas → [[parking-lot]]

Anti's words: *"I don't trust myself in patiently building one by one. I get excited and jump here and there."*
